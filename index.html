Configurar base de datos

Conectá tu proyecto de Supabase para que los datos persistan.
Es gratis y tarda 5 minutos.

1
Entrá a supabase.com y creá una cuenta gratis. Luego creá un nuevo proyecto.
2
En tu proyecto, andá a SQL Editor y ejecutá el script del archivo supabase-schema.sql que está en el repo.
3
Andá a Project Settings → API y copiá la Project URL y la anon public key.
4
Pegá los valores abajo y guardá.
